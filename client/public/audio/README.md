# Audio Files

Placez votre fichier de musique de fond ici nommé `background-music.mp3`.

Pour ajouter de la musique de fond à votre application :

1. Obtenez un fichier audio MP3 (musique traditionnelle arabe ou instrumentale)
2. Nommez-le `background-music.mp3`
3. Placez-le dans ce répertoire

Le lecteur de musique tentera de charger ce fichier automatiquement. Si le fichier n'est pas présent, l'application fonctionnera normalement sans musique.

## Suggestions de musique

- Musique instrumentale arabe traditionnelle
- Oud ou Qanun instrumentaux
- Musique d'ambiance orientale
- Sons de nature du Moyen-Orient

Assurez-vous d'utiliser de la musique libre de droits ou dont vous avez les droits d'utilisation.
